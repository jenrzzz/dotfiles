# tips.txt — personal workflow tips, shown at login by `tips` (see scripts/bin/tips).
#
# Format: records separated by a line containing only `%`. Each record's first
# line is `category: <name>`; the rest is the tip body (1-2 lines is ideal).
# Lines beginning with `#` here are just comments to humans — they are part of a
# record's body only if you put them inside one, so keep them at the very top.
#
# Grow this file with `/tips-gen` (reads your configs and appends fresh tips).

category: tmux
C-a J — fuzzy jump to or create a session, over running sessions + project dirs + zoxide history.
%
category: tmux
C-a M moves the current window to another (or brand-new) session; C-a P merges other windows into this one as tiled panes (TAB to multi-select).
%
category: tmux
C-a R refactors a bloated session into one session per directory basename. Nothing ever moves on its own — it's on-demand only.
%
category: tmux
A session with 6+ windows flashes an amber ⚠ in the statusline — your cue to hit C-a R.
%
category: tmux
Panes are vim keys: C-a h/j/k/l selects left/down/up/right, C-a a cycles to the next pane.
%
category: tmux
C-a z zooms the current pane fullscreen (toggle) — focus on one thing, then pop back.
%
category: tmux
C-a + arrow resizes the current pane and is repeatable: hold prefix once and keep tapping.
%
category: tmux
C-a ( / ) step prev/next session (repeatable); C-a g toggles back to the last session.
%
category: tmux
C-a C-h / C-a C-l step prev/next window (repeatable); C-a r toggles the last window.
%
category: tmux
C-a ^T splits a 25% horizontal pane — handy for a small status or log pane.
%
category: tmux
The 👻 window (htop) and 😍 window (mutt) are mirrored into every session and always left in place.
%
category: bash

%
category: bash
sbq fuzzy-picks a branch and checks it out; bq just previews branches with delta.
%
category: bash
qq fuzzy-picks files with a bat preview; vq does the same and opens the picks in nvim.
%
category: bash
Ctrl-R is rebound to hq: fuzzy-search your eternal history and run the chosen command.
%
category: bash
cdq fuzzy-cd's into any subdirectory up to 4 levels deep.
%
category: bash

%
category: bash
mkd <dir> makes a directory and cd's into it in one step.
%
category: bash
scratchd spins up a throwaway dir (and changes your prompt); unscratch -r cd's back and deletes it.

category: bash
`c` trims the trailing newline and copies stdin to the clipboard — e.g. `pwd | c`.

category: bash

%
category: bash
server [port] serves the current dir over HTTP (default 8000) and opens it in the browser.

category: bash

category: bash

category: bash

category: nvim

category: nvim

category: nvim

category: nvim

%
category: nvim

%
category: nvim

%
category: nvim

category: nvim

category: nvim
